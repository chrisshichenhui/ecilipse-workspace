公司简介

"爱睿智健康科技(北京)有限公司"于2016年01月26日成立。

法定代表人卓志隆 ，公司经营范围包括：技术推广服务；健康管理（须经审批的诊疗活动除外）；医院管理（不含诊疗活动）；企业管理；企业策划；健康咨询（须经审批的诊疗活动除外）；

经济贸易咨询；计算机系统服务；会议服务；组织文化艺术交流活动（不含演出）；承办展览展示活动；设计、制作、代理、发布广告；货物进出口、技术进出口、代理进出口；维修计算机；

销售计算机、软件及辅助设备、日用品、电子产品、化工产品（不含危险化学品）、机械设备、仪器仪表、文具用品、I、II类医疗器械、日用品、消毒用品、化妆品；应用软件服务（不含医用软件）； 软件开发；互联网信息服务；保险代理业务；销售第Ⅲ类医疗器械等。
